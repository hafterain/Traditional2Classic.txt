# Traditional2Classic.txt

## 简介

在一切开始之前让我们一起念：

> stupid Unihan, fuck Unicode.

字源分离原则实在是太蠢了。就算你自己用了一点明体这样的把新字形码位直接映射到旧字形的字体，没安装这类字体的人们依旧要看到新字形。

该项目提供一套一对一的转换表，期望实现「能显示旧字形的，尽可能显示旧字形」。
